# Anti-AI Aesthetic Guidelines

## Core Philosophy
ArsonPixelz is a human-led creative monolith. All messaging must reflect agency, intentionality, and high-end editorial design. Avoid generic "robotic" tropes that imply automated, shallow generation.

| :--- | :--- |
| "Autonomous System" | "Creative Monolith" |
| "Protocol", "Intel", "Data" | "Manifesto", "Vision", "Blueprint" |
| "AI-Driven", "Algorithmic" | "Engineered", "Executed", "Human-Led" |
| "Processing...", "System Online" | "Active", "Igniting", "Live" |
| "Neural", "Cyber", "Quantum" | "Industrial", "Architectural", "Tactile" |

## Design Rules
1. **No Nuanced Small Text**: Avoid tiny font-size labels like "System V1.0.4". If it's important, make it bold and clear.
2. **Big & Unique**: Use massive typography and oversized icons. Small, fiddly bits feel like automated templates; large, deliberate choices feel human.
3. **Contrast**: Lean into high contrast (Black/White/Arson Orange). Avoid "tech-blue" or "glowing neon" aesthetics associated with generic AI.
4. **Editorial Layout**: Think high-end fashion or industrial design magazines, not software dashboards.
5. **Typography Unification**: Use the unslanted **'Anton', sans-serif** font for all primary section headers. Avoid italics or slants unless specifically requested for sub-labels. This font represents the "Creative Monolith" stability.
